A mod manager for Avatar: The Game (2009)

## Features

- **Load Order Management** - Buttons to set merge priority
- **Enable/Disable Mods** - Toggle mods on/off without removing them
- **File Viewer** - Browse and preview mod contents before merging
- **Batch Merging** - Merge multiple mods into a single `patch.pak`
- **Backup & Restore** - Save and restore original game files

## Usage

1. Add your mod `patch.pak` files using the "Add Mod" button
2. Arrange mods in load order **(top mods override bottom mods)**
3. **Enable/disable** mods with checkboxes
4. Click "Merge Mods" to create the final `patch.pak`
5. Find your merged file in the `merged` folder
